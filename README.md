# smart_clinic

## pembagian tugas

### visual javafx
tugas utama:
- membuat tampilan menggunakan javafx dan fmxl
- mendesain tampilan agar enak dipandang dan mudah digunakan
- menghubungkan setiap component berdasarkan id, action, dan controller sesuai dengan file controller yang dibuat 
### controller
- membuat controller untuk menghubungkan bagian frontend dan backend
- mengatur supaya controller mudah dipahami oleh kedua bagian
### DAO dan service
- membuat skema tabel di MySQL sesuai dengan rancangan
- membuat service agar lebih mudah digunakan di controller
- membuat DAO sebagai interface antara java dengan SQL
